Classes

  ClassDescription
EnemyTank
The EnemyTank class. High health and Armor. Uses Slug Ammo.
EnemyTruck
The EnemyTruck class. Low Health and Armor. Uses Missle Ammo.
Entity
Abstract Base class for all character agents (AKA Entities). Every Entity has Health, Armor, and MaxHealth properties. Every Entity has Attack and Defend methods.
ExplodeAmmo
ExplodeAmmo is a type of Weapon. Has no bonus to attack. Inpercise (wide range) but high damage.
GameObject
Abstract Base Class for all other classes. Gives every inherited class the Name property.
MissleAmmo
MissleAmmo is a type of Weapon. Has high bonus to attack. Average damage spread with high damage.
PiercingAmmo
PirecingAmmo is a type of Weapon. Has moderate bonus to attack. Average damage spread with low damage.
Player
The Player class. High Health and Armor. Uses various kinds of Ammo selected by the player.
Program
Main class of the program. Contains the Entry Point.
SlugAmmo
SlugAmmo is a type of Weapon. Has low bonus to attack. Percise but moderate damage.
Weapon
Abstract base class for all weapons in the game.
